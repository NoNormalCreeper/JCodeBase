## What is it
a simple codebase.includes many utils wrote in cpp,java etc.
## How to use
run these commands in windows:
``
gradlew build
mkdir native\build
cd native\build
cmake ..
``
Then use any of ides you like to compile it.  
Put it in the root dir and named it as 'native.dll'.  
Run tests.
